# ToDo-List-Application

A modern, local-first Android application designed to help students organize their academic life. Built with Jetpack Compose and Room Database, this app demonstrates the implementation of full CRUD operations with a clean, reactive user interface.

Features:

Persistent Storage: Uses the Room Persistence Library to ensure tasks are saved locally and survive app restarts.
Reactive UI: Built entirely with Jetpack Compose for a smooth, state-driven user experience.
CRUD Operations: Users can create, read, update, and delete tasks seamlessly.
Background Processing: Utilizes Kotlin Coroutines to handle database operations off the main thread, ensuring zero lag.
Modern Architecture: Follows the MVVM (Model-View-ViewModel) pattern for scalability and maintainability.


Tech Stack:

Language: Kotlin
UI Framework: Jetpack Compose (Material 3)
Local Database: Room Persistence Library
Asynchrony: Kotlin Coroutines & Flow
Architecture: MVVM
Dependency Management: Gradle (Version Catalog)
